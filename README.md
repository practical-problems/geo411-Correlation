# geo411-Correlation
An analysis of correlation
# Title:
An analysis of correlation of between life expectancy and GDP per capita of different countries.
# Overview: 
This analyse the correlation between life expectancy and GDP per capita of various countries around the world based on data provided by the World Bank organization for the year 2023. 
# Method:
The correlation is calculated through the use of R Studio using Pearson's test for correlation and Spearman's test for correlation.
# Keyfinding: 
Spearman's test has the output coefficient of 0.82, signalifying a strong monotonic relationships between life expectancy and GDP per capita. While, Pearson's test has the coefficient output of 0.72, which shows there is a strong linear positive relationship between GDP per capita and life expectancy.
# Tools:
R Studio, Github and Microsoft Word
